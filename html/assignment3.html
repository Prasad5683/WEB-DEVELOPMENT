HTML Formatting & Heading Tags

This page demonstrates commonly used HTML tags for text formatting and headings.

Heading Tags

<h1> Heading 1 (Most Important)

<h2> Heading 2

<h3> Heading 3

<h4> Heading 4

<h5> Heading 5
<h6> Heading 6 (Least Important)

Headings help structure your content and improve readability and SEO.

Text Formatting Tags

<b> - Bold

This text is bold.

<strong> - Strong Importance

This text is strongly emphasized (bold with meaning).

<i> - Italic

This text is italicized.

<em> - Emphasized Text

This text is emphasized (italic with meaning).

<u> - Underlined

This text is underlined.

<mark> - Marked/Highlighted

This is highlighted text.

<small> - Smaller Text

This is smaller text.

<del> - Deleted Text

This is deleted text.

<ins> - Inserted Text

This is inserted text.

<sub> - Subscript

H2O (subscripted 2)

<sup> - Superscript

x2 = 25 (superscripted 2)

These tags are essential for semantic, accessible, and well-structured webpages.
